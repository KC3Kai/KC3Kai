What do these sections do?
  • In Current Goal section you can view and edit all current leveling goals.

  • Goal Template allows you to

    • auto-apply some leveling settings when new leveling ship is added.
    • filter ships by their types
    • apply templates to existing leveling goals

  • Recommended and Other Ships list all your heartlocked ships not listed in Current Goals. If one of your ships is not yet reaching her remodel level, she will show up in Recommended section with target level set to the minimal remodel level. Otherwise she will appear in Other Ships section with more general goals (Level 99 or Level 175).

Where has map selector gone? What is Base Exp?

Since game phase 2 has changed the experience distribution of normal maps, now exps gained from battles are different by enemy composites, no longer the same value by maps.

So instead of choosing a map to get the mapped Base Exp, now you can directly input the expected (maybe average) Base Exp value of the node(s) from where you are going to grind. Base Exp value can be found at right-top of in-game battle result screen, or the Node EXP after expanding Nodes in your Strategy Room Maps History page.

How goal template works?

When adding a ship leveling goal, all enabled templates are attempted in order and the first template that has a matching ship type will be applied to that ship.

For example, assume if you have the following templates in order:

  1. CL,DD with map 1-5 and some other settings
  2. CL,DD,CAV with map 5-4 and some other settings
  3. Any with map 3-2 and some other settings

The first template matches ship type "light cruisers(CL) or destroyers(DD)"; the second template matches ship type "light cruisers(CL) or destroyers(DD) or aviation cruisers(CAV)"; and the third one matches any ship type.

When a destroyer is added to leveling goal, despite all 3 templates match her ship type, only the first template is applied, her map will be set to 1-5 and so does other leveling settings. When an aviation cruiser is added, the second template is applied because this is the first one that matches her ship type. Similarly when a battleship is added, the third template is applied.

How to use goal templates?
  • "New template" link create a new template with lowest priority (at the end of the list)
  • click green buttons (up / down) to change template orders.
  • click "Enabled" / "Disabled" to toggle a template (a disabled template won't be applied automatically)
  • "X" removes a template, "E" changes a template to edit mode, "S" saves changes.
  • "Apply to Current Goals" does what its name suggests. Before template application, it will first pop up a confirm message listing all ships in current goals it can affect.
  • "Highlight Coverage" highlights all ships it can match. "Clear Highlight" can then be used to clear highlights. (Note that even if a template filter highlights a ship, that doesn't mean this template will be auto-applied to that ship, either because other templates have higher priority or this template is disabled)
Tell me more about ship type filters!

The ship type filter is a comma-separated list of ship types, for now only English abbrs and *(will be explained shortly) are accepted (case-insensitive), following inputs are all valid:

  • cl, dd,ca
  • CL, DD, CA, DD, CA
  • *
  • *, ca, dd

Don't worry about case or putting extra spaces or having duplicated stuff when editing, they can be processed properly.

The English abbreviations of ship types is also used in "Ship List" tab of strategy room: setting ship type filter to CL,DD,CA selects exactly the same list of ships as only turning on DD, CL, CA filters in "Ship List" tab.

* is a wildcard that matches all ship type, and any ship type filter containing * matches all ships. (Note that despite * is shown as Any in view mode, Any is an invalid ship type and you should always use * when editing.)

You can keep a template with ship type * at the end of the template list, to serve as a "default setting".
