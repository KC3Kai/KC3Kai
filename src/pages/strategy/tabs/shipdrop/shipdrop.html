Are these records from a data collecting cloud service?
Nope. These drops are from recorded sortie history data of your own, if you have enabled the history recording.
It's just an alternative view of Maps / Events History pages, no remote data from any report server have been fetched.
It's expected that unable to determine boss node from 'ancient' history or events data.
Will be a little weird if node letters are not ready or your history data have been partially corrupted.
Will some ship types listed here drop?
Nope. All available ship types are just copied from Ship List page.
To activate only 1 filter quickly instead of switching, like doing it on the Ship List page, you can hold Alt key and left click one of them.
