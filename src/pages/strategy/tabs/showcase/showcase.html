Some equipment are missing!
Try to refresh the page and it will fix itself.
How do I screenshot?
You can use lightshot, gyazo, or puush or any preferred screenshot software.
The top 6 here is different from the top 6 on Ship List!
Those are top 6 of the same levels. Unfortunately we will not go for the extra effort in further sorting them.
What are the colors and digits under each ship?
Firepower, Torpedo, AA, Armor, and Luck. If it's a colored check, it's maxed. If it's a number, it is how far you are from maxing it. The exception is Luck which shows your current value but is colored.
I can't see anything!
First thing to try is just refresh the page. If that does not work, check if you have already played the game with the F12 panel open. It is the only way for us to know your data.
I want my Radars sorted by LoS!
I am sorry, we can only sort each category by one attribute at a time.
Night scout seaplane should be on the Night gear box!
Unfortunately, in game data it is still categorized under sea planes. I am sorry but we will not go for the extra effort of making an exception for this equipment.
Why some of my equipment\ships are not shown on export image?
Image are showing only locked ships and items. If you acquired and locked item in current game session, you might need to go to home port once.
Exporter is downloading image instead of opening it in new tab or Imgur!
In case of Imgur error or image data string is being too big for new tab to handle it will be downloaded.
How does export event locking tags work?
If ship is locked in some operation you will see colored background behind it's name. If you only plan to lock it there you will see colored circle right after ship level. Please note colors on image doesn't necessarily match lock colors in game. You could compare them here.
