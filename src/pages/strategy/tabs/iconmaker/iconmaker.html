Basic Usages:
  • To start up, input the ID of a shipgirl first, her image URL will be auto generated.
    • Auto generated image URL from http://fleet.moe by default, also using its seasonal index.
    • Will generate game official image URL (from your server) if Official checkbox enabled.
    • Can paste any cross-site referable image URL to replace auto generated one.
    • Should enable Abyssal checkbox beofre inputing ID for an Abyssal ship.
  • To cancel image loading or clear all existed icons, refresh this page.
  • To crop a face icon, adjust coordicate and zoom by aligning her hair edges, eyes, lower jaw and etc.
  • To remove a reference or cropped icon, left click on it.
  • To save a cropped icon, use `Save image as` in its context menu.
