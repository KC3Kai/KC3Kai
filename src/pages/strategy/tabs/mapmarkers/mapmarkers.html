Basic Usages:
  • Input the world and map ID wanna to be viewed;
  • Enabling All Nodes will show unlocked node detials if map data is ready (have sortied to this map once);
  • Enabling Edge IDs, Arrows, Red BG will add optional details to map rendering;
  • Enabling Enemies will show offical in-game abyssal ship icons (still no submarine icon for some nodes);
  • Input / paste makers JSON, both markers and JSON texts will be updated on focus out;
  • Drag markers to their proper position on map. Zoom to 100% if accurate coordinates needed;
  • To remove marker, delete JSON directly or drag it out of map.
How to show hidden nodes & routes before unlocked in game?
Enabling the data digging flag will do, execute this code in console:
KC3StrategyTabs.mapmarkers.definition.digEventSpots = true
NOTE: Will try to fetch JSON data from official server until 404 is responded.
